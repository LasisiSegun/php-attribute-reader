## 📂 What Are PHP Attributes?

PHP attributes are a way to add extra information to your PHP code. Think of them as labels you stick on your code to explain or change how it works.

For example, attributes can help describe what a function does or tell the program to do something special with a piece of code. php-attribute-reader helps you find these labels quickly and clearly.
